* Goal

Implement **dynamic policy-driven field filtering** without:

- polluting Domain layer
- duplicating authorization logic
- introducing GraphQL complexity
** breaking Clean Architecture

We want:

#+begin_quote
Authorization engine decides WHICH FIELDS are visible.
Application layer projects entity accordingly.
#+end_quote

Result:

- dynamic policies (DB/config driven)
- role-based + contextual filtering
- reusable across all use cases

* Step 5 — Projector (Core Behavior)

Create:

**user_projector.rs**

#+begin_src rust
pub struct UserProjector;

impl UserProjector {
pub fn project(
user: &User,
access: &UserAccess,
) -> UserView {

```
    let allow = |f| access.allowed_fields.contains(&f);

    UserView {
        id: allow(UserField::Id)
            .then(|| user.id().clone()),

        email: allow(UserField::Email)
            .then(|| user.email().clone()),

        username: allow(UserField::Username)
            .then(|| user.username().clone()),

        profile: allow(UserField::Profile)
            .then(|| user.profile().clone()),

        preferences: allow(UserField::Preferences)
            .then(|| user.preferences().clone()),

        status: allow(UserField::Status)
            .then(|| user.status().clone()),

        failed_logins: allow(UserField::FailedLogins)
            .then(|| *user.failed_logins())
            .flatten(),

        locked_until: allow(UserField::LockedUntil)
            .then(|| user.locked_until().clone())
            .flatten(),

        last_login: allow(UserField::LastLogin)
            .then(|| user.last_login().clone())
            .flatten(),

        version: allow(UserField::Version)
            .then(|| *user.version()),
    }
}
```

}
#+end_src

Domain untouched.

* Step 6 — UseCase Changes

#+end_src

pub async fn execute(
&self,

#+end_src

Implementation:

#+begin_src rust
let user = self.repo.get(id).await?;

let access =
self.authz.evaluate_user_access(&ctx, &user);

if !access.can_read {
return Err(Error::Forbidden);
}

let view = UserProjector::project(&user, &access);

Ok(view)
#+end_src
