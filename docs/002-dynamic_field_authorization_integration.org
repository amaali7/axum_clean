* Purpose

This document describes **exact repository changes** required to integrate:

- Dynamic field authorization
- Authorized projection
** Policy-driven data shaping

WITHOUT breaking existing architecture.

This is an *integration guide*, not a design explanation.

---

** High-Level Dependency Rule

We MUST preserve Clean Architecture direction:

#+begin_example
domain
↑
application
↑
interface (axum)
↑
infrastructure
#+end_example

Authorization is an **application service**, not domain logic.

Projection lives in application layer.

* Step 6 — Projector

projector.rs

#+begin_src rust
pub struct UserProjector;

impl UserProjector {
pub fn project(
user: &User,
access: &UserAccess,
) -> UserView {
let allow = |f| access.allowed_fields.contains(&f);

```
    UserView {
        id: allow(UserField::Id)
            .then(|| user.id().clone()),
        email: allow(UserField::Email)
            .then(|| user.email().clone()),
        username: allow(UserField::Username)
            .then(|| user.username().clone()),
        profile: allow(UserField::Profile)
            .then(|| user.profile().clone()),
        preferences: allow(UserField::Preferences)
            .then(|| user.preferences().clone()),
        status: allow(UserField::Status)
            .then(|| user.status().clone()),
        failed_logins: allow(UserField::FailedLogins)
            .then(|| *user.failed_logins())
            .flatten(),
        locked_until: allow(UserField::LockedUntil)
            .then(|| user.locked_until().clone())
            .flatten(),
        last_login: allow(UserField::LastLogin)
            .then(|| user.last_login().clone())
            .flatten(),
        version: allow(UserField::Version)
            .then(|| *user.version()),
    }
}
```

}

* Step 7 — UseCase Integration

OLD:

#+begin_src rust
let user = repo.get(id).await?;
Ok(UserResponse::from(user))
#+end_src

NEW:

#+begin_src rust
let user = repo.get(id).await?;

let access =
authz.evaluate_user_access(&ctx, &user);

if !access.can_read {
return Err(Error::Forbidden);
}

let view =
UserProjector::project(&user, &access);

Ok(view)
#+end_src
